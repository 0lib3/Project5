# Project 5


## Implementation
Made in Unity Editor 2022.3.17f1

A silly FPS maze in which you are actively being pursued by balls of cheese.

- First person camera view
- Blaster gun
- Randomly spawned enemies
- Procedurally generated maze map
- Fun music
- Timer
- Start/End game
